# Genius - Educational CTF Game for Kids

Genius is a browser-based Capture The Flag (CTF) game designed for children aged 10-12. The game teaches cybersecurity concepts through fun challenges and puzzles, with Einstein as a friendly guide.

## Features

- Multiple challenge categories:
  - Encryption puzzles (Caesar, ROT13, Base64)
  - Interactive puzzles
  - UI challenges
  - Cybersecurity quiz
- Bilingual support (Dutch/English)
- Dark mode support
- Progress tracking
- No DevTools required
- Chromebook-friendly

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Project Structure

- `src/app/` - Next.js app router pages
- `src/components/` - React components
- `src/lib/` - Utility functions and game logic
- `public/` - Static assets

## Technologies Used

- Next.js 14
- TypeScript
- Tailwind CSS
- Framer Motion
- React Icons
